Portfolio Next.js + Tailwind

Installation
1. `npm install`
2. `npm run dev`

Modifier les expériences
- Éditez `src/data/experiences.json`.
- Champs: `id`, `poste`, `entreprise`, `lieu`, `debut` (YYYY-MM), `fin` (YYYY-MM ou "Présent"), `missions` (array), `tech` (array).

Personnaliser
- Métadonnées: `src/app/layout.tsx` (`metadata`).
- En‑tête: `src/components/Header.tsx`.
- Pied de page: `src/components/Footer.tsx`.
- Sections: `src/app/page.tsx`.
